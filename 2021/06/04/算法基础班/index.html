快排

分治的思想

  1. 确定分界点：q[l]，q[r]，q[(l+r)/2]
  2. 调整区间：[<=x, x, >=x]，x 左侧都是小于 x，右侧都是大于 x
  3. 递归处理左右两段

归并

分治的思想

  1. 确定分界点：mid = (l + r)/2
  2. 递归排序 left，right（区间分为 left 和 right 两部分，均分）
  3. 归并，合二为一

整数二分

有单调性的一定可以二分，但可以二分的不一定有单调性

二分的本质：边界，两侧逼近满足不同性质的分界点

1.红色点
mid = l + r + 1 >> 1
if (check(mid)（是否满足红色的性质) [mid, r], l = mid
else [l, mid - 1], r = mid - 1

2.绿色点
mid = l + r >> 1
if (check(mid)（是否满足绿色的性质) [l, mid], r = mid
else [mid + 1, r ], l = mid + 1

01 背包

动态规划（从集合的角度来理解）

  • 状态表示 f(i, j)
    • 集合
      • 所有选项
      • 条件
        • 只从前 i 个物品中选
        • 总体积 ≤ j
    • 属性，Max，（Min，元素的数量）
  • 状态计算——集合的划分
    • 不包含 i 的：f(i-1, j)
    • 包含 i 的：f(i-1, j - vi) + wi
    • 两者取一个最大值

完全背包

  • 状态表示 f(i, j)
    • 集合：所有只考虑前 i 个物品的，且总体积不大于 j 的所有选法
    • 属性，Max，Min，元素的数量
